Otimização Do Pc

- Troque o navegador por Opera e desative o outro tecla Windows-Painel de Controle-programas e recursos-ativar ou desativar recursos-Marcar Internet Explorer
	- Instale Uma Extensão de tradutor e tbm de baixar videos do youtube.
- Não Instalar Anti-Virus O Proprio Sistema Ja Trabalha Com Windows Defender
- Desativar Serviços XPS é um formato criado para criar, imprimir e visualizar arquivos no computador. uma alternativa ao Adobe PDF, raramente é utilizado.
- Desativar Hospedagem no serviço de ativação do processo ou (WAS - Windows Process Activation Service) 
	Criado para ajudar o usuário a desenvolver aplicativos e serviços,o WAS foca em quem planeja transformar o seu computador em um servidor caseiro.
- Desativar Remote Differential Compression (RDC) o RDC é um serviço mais voltado para ambientes de servidor. 
	Permite a conexão remota entre dois computadores, caso do Remote Desktop Connection ou semelhante.
- Desativar Cliente de pastas de trabalho, Links de pastas compartilhadas na rede 
	são bastante comuns em ambientes corporativos, mas pouco comum em PCs domésticos. Se você não possui pastas remotas em casa, pode desativar este serviço.
- Desativar Windows PowerShell, o PowerShell é uma ferramenta semelhante ao Terminal do Linux para a automação de tarefas e criação de scripts. 
	É uma espécie de “Super CMD”, mas pode se desativado caso não seja utilizado.
- Desativar Windows Fax and Scan, trata-se do subsistema para lidar com aparelhos de fax e scanners. Se você não usa aparelhos do tipo, pode desativá-lo sem medo
- Desativar Microsoft Print to PDF, Alguns sites oferecem a opção de imprimir certos arquivos, e este recurso
	permite que essa “impressão” seja virtual, salvando o documento no formato PDF.
- Desativar Internet Printing Client, Ainda falando de impressão, este recurso permite a impressão via internet dentro da rede local. 
	Se você não possui uma impressora que funciona na rede, pode desativar este recurso sem problemas.
- Desativar Windows Media Player, Ainda no departamento de softwares sobressalentes, temos o Windows Media Player. Incapaz de rodar formatos mais modernos de arquivos, 
	ele pode ser substituído por excelentes alternativas gratuitas, como é o caso do VLC Media Player.
- Desativar DirectPlay, Componente da API do DirectX, o DirectPlay é utilizado para rodar jogos bastante antigos, 
	dificilmente jogados atualmente. Você pode desativá-lo em Componentes legados ->DirectPlay.
- Desativar acesso Remoto, Painel de Controle\Todos os Itens do Painel de Controle\Sistema\Configurações Remotas Desmarcar Opção.
- Desativar atualizações do Windows Update, Mas Sempre q der, Verificar se há atualizaçoes, Painel de Controle\Todos os Itens do Painel de Controle\Windows Update\Alterar configurações
- Gerenciador de Tarefas\ Inicialização\ Desabilitar apenas os programas que não necessitam iniciar ao ligar
